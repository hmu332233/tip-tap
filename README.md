# Tip-Tap

## What's Tip-Tap?

Tip-Tap은 손가락만 있으면 할 수 있으면 할 수 있는 간단한 게임의 모음입니다.  
친구들과 간단한 내기, 팀 나누기, 보드게임 순서 정하기등 다양한 상황에서 활용해보세요!
